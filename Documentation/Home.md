# Overview
Astro Utils is a package developed by me ([AstroTurffx](https://github.com/AstroTurffx)) to aid me and other people create better games in unity faster. There isn't a lot of things in this package but I will definitely add more things in the future when I think of anything to add.

# Package contents
This package currently only contains scripts but this might change in the future.

# Installation instructions
Unity has good [documentation](https://docs.unity3d.com/Manual/upm-ui-giturl.html) on how to do this.

# Requirements
This package should work with most newer unity versions (I developed mine on unity 2021.3) and will work on most pcs
